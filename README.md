# Disk-Scheduling-Visualizer
FCFS, SCAN and C-SCAN Disk Scheduling Simulator

## Course
Operating System

## Overview

Disk Scheduling Visualizer is an Operating Systems project developed in C++ to simulate and compare disk scheduling algorithms. The project calculates disk head movement, total seek time, and provides a web-based visualization of algorithm execution.

## Algorithms Implemented

* FCFS (First Come First Serve)
* SCAN (Elevator Algorithm)
* C-SCAN (Circular SCAN)

## Features

* Menu-driven C++ application
* Disk request scheduling simulation
* Seek time calculation
* Step-by-step head movement tracking
* Interactive browser visualization
* Algorithm performance comparison
* JSON-based data transfer between C++ and web interface

## Technologies Used

* C++
* HTML
* CSS
* JavaScript
* JSON

## How It Works

1. The user enters disk size, head position, direction, and request queue.
2. The selected algorithm processes the requests.
3. Results are exported to JSON files.
4. JavaScript reads the JSON data and displays visual graphs and comparisons in the browser.

## Learning Outcomes

* Understanding disk scheduling algorithms
* Comparing seek time performance
* File handling in C++
* JSON data exchange
* Integration of C++ with web technologies
